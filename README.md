#caltrain
---
###Usage:
```bash
$ caltrain sv south next
# => 22:53:00 - Train 194 (local)
#  *sv -> law -> sc -> sj -> tam

$ caltrain pa south list
# => 23:36:00 - Train 196 (local)
#    *pa -> cal -> sa -> mv -> sv -> law -> sc -> sj
#  24:57:00 - Train 198 (local)
#    *pa -> cal -> sa -> mv -> sv -> law -> sc -> sj
```
